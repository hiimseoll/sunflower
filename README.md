# sunflower 해바라기

<img src = "imgs/fanGif.gif">

온도가 높은 곳으로 바람을 보내주는 탁상용 선풍기.



사용 부품:
- AMG8833
- N20 Motor * 2
- BLDC Motor
- DC/DC StepUp Module
- 3000mAh LiPo Battery
- 5V/2A Battery Charging Module
- 0.96 OLED Display
- L298N
